使用Wordpress创建个人博客

之前使用的是Hexo + gihub搭建的博客，但是有很费劲的问题：
1、费劲，不能跨机器使用，每次换一台机器都必须得重新装一下环境；
2、百度爬虫爬不到，只能Google搜索到。
Wordpress：
1、方便部署，登录后台管理器，随时随地写文章管理网站。

准备工作

  • 买一个vps

UFO VPS，香港的vps，可以选择香港、美国的服务器，如果想搭建vpn的话，相对稳定一点。我买的是这个。
linode，日本的vps，也可以。
搬瓦工，比较便宜的vps。

  • 买个域名，几十块一年就行了。可以去阿里云、腾讯云、美橙互联等上面买。

3、登录你的vps ssh root@1.1.1.1，后面的是你的vps的服务器地址，然后回车，输入你的密码登录进去。

4、一键安装lnmp，如果安装成功了，打开你的vps地址，如果出现如图所示，则就算安装成功了。

5、安装pureftp。File zilla

6、添加网站，只需要看第一步的##添加网站(虚拟主机)##这一部分即可。

7、cd /home/wwwroot/www.auu.space/，这是上一步设置的网站资源地址，这里的后面是我的网站，你需要输入你的网址。

8、安装Wordpress，先wget https://cn.wordpress.org/wordpress-4.7.4-zh_CN.tar.gz下载，然后tar xzvf wordpress-4.7.4-zh_CN.tar解压。

9、把资源移动一下 cp -R wordpress/* ./，把wordpress下的资源全部都移动到当前的网站目录下。

10、然后添加一下文件夹的权限 chmod -R 777 /home/wwwroot/www.auu.space，主要是安装插件、主题什么得需要/home/wwwroot/****/wp-content网站的这个目录下的操作权限。

11、然后配置nginx，vim /usr/local/nginx/conf/nginx.conf，找到server下的root的值为你的网站资源的目录，如我的/home/wwwroot/www.auu.space;，然后保存

12、nginx -s reload，重新加载配置，nginx -c /usr/local/nginx/conf/nginx.conf，重启nginx。nginx操作命令

13、安装你的Wordpress，打开安装路径http://..你的vps地址../wp-admin/setup-config.php，跟着提示一步步的填写内容即可。

14、这时候打开你的vps的地址就会出现你的网站内容。

15、这时候已经算是好了，进到你购买的域名服务商的后台，添加你的域名A记录解析到你的vps地址。

16、打开wordpress的管理目录http://...你的网站.../wp-login.php，在这里你可以管理你的个人网站。

17、如果习惯了markdown，可以安装WP Editor.md插件，知乎讨论。

18、习惯了hexo的文章概要的话，wordpress里也可以使用<!--more-->来分割，注意，hexo里的是<!-- more -->，这里多了两个空格，如果不想用这个的话可以看看这个教程WordPress 显示文章摘要方法 详细篇。

19、linux 命令、 scp 命令、pip命令

20、wordpress安装插件、主题、SEO优化。
